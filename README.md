PagerDuty Notifier
==================

Provides [PagerDuty](https://www.pagerduty.com) integration for Symfony Notifier.

DSN example
-----------

```
PAGERDUTY_DSN=pagerduty://TOKEN@SUBDOMAIN
```

where:
 - `TOKEN` is your PagerDuty API token
 - `SUBDOMAIN` is your subdomain name at pagerduty.com
